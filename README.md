### Simple Blog
A simple blog application using HTML, CSS, and JavaScript to create, edit, and delete posts. The posts are stored in the browser's local storage.
## Features
-Add new blog posts
-Edit existing blog posts
-Delete blog posts
-Posts are saved in local storage
